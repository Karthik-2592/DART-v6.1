# DART - Full-Stack Music Application

Welcome to DART, a full-stack music application that allows users to manage their music experience seamlessly. Equipped with robust features, DART provides an innovative way to enjoy, manage, and interact with music. 

## Features

### Account Management
- User registration and authentication.
- Profile management to customize user experience.

### Built-in Visualizer
- Stunning visual effects synchronized with the music being played.
- Customizable visual themes.

### Comment System
- Users can comment on songs and playlists.
- Threaded comments for meaningful conversations.

### Follower System
- Follow your favorite artists and friends.
- Stay updated with new releases and playlists shared by followers.

### Playlist Management
- Create, edit, and delete playlists.
- Share playlists with friends or make them public.

### Song Management
- Upload songs and manage your music library.
- Add metadata for better organization and search functionality.

## Tech Stack
- **TypeScript** (66.8%) - For type-safe programming and improved developer experience.
- **JavaScript** (21.2%) - For dynamic content and interactivity.
- **PLpgSQL** (6.2%) - For efficient database operations.
- **CSS** (2.8%) - For styling the application.
- **Python** (2.7%) - For backend services and logic.
- **HTML** (0.3%) - For basic structure of the web application.

## Getting Started
To get started with DART, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/Karthik-2592/DART-v6.1.git
cd DART-v6.1
npm install
```

Then you can start the application:

```bash
npm start
```

For further documentation, please refer to the [Wiki](#) for in-depth guides and tutorials on how to make the most out of DART!
